# NoShowIQ

A prediction API for medical appointment no-shows.

## Live API Endpoints

- `GET /health` — Health check
- `POST /predict` — Predict no-show risk
- `GET /history` — Last 20 predictions
- `GET /stats` — Aggregated stats
